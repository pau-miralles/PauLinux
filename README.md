# PauLinux
Personal **NixOS** setup for Framework 13 AMD, with *SwayFX, Waybar, Stylix, Neovim, etc*

---
### Installation
1. Connect to internet (ethernet or wifi using `nmtui`)
2. Curl and run the script:
```bash
curl -L https://raw.githubusercontent.com/pau-miralles/PauLinux/main/install.sh | bash
```
