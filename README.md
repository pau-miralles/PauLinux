# PauLinux
Personal **NixOS** setup for Framework 13 AMD with *SwayFX, Waybar, Stylix, Neovim, etc*

---
### Installation
1. Connect to internet (ethernet or wifi with `nmtui`)
2. Enable git on the shell: `nix-shell -p git`
3. Download and run the script:
```bash
curl -O https://raw.githubusercontent.com/pau-miralles/PauLinux/main/install.sh
chmod +x install.sh
./install.sh
```
4. Reboot: `reboot`
